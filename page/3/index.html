解放你的数据线——AirTrans

发表于 2014-02-06   |   分类于 Coding

很高兴的在女朋友的想法之下，通过一上午的努力做出了这样一个用来摆脱数据线的传输软件

该软件可以使用局域网环境实现文件的传输、删除以及重命名，其过程自动识别，自动给出FTP连接地址

由于本人还没有能够独立作出这种软件的能力，该软件是基于MiFileExplorer中提取后修改，在此感谢前辈们所做的努力
