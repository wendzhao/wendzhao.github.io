网盘搜索神器---KUMA网盘搜索

发表于 2014-07-01   |   分类于 notice   |  

好久没有做出任何像样的东西出来了，整个人也因为最近发生了太多的事情而勉强支撑着。或许我的人生就像在做过山车，一会高一会低的，很压抑内心。人生中的最后一个暑假，该来的都来吧，该走的也走吧，该学会独立了。

还是说说刚写完的这个工具吧，整个出发点还是好的，因为现在大多数人分享资料的方式不外乎网盘，这样的话搜索学习资料就会方便一点。博主用的是百度的引擎，因为家里谷歌没法用（还是最近大家都上不去？），有一个弊端就是百度最近网盘的外链都在清理当中？好像用百度搜自家网盘出来的结果并不多。如果有人用的话，可以以后考虑使用bing或者yahoo的引擎。

我知道这款网盘搜索器还是有弊端的，就是没法把所有网盘的搜索结果聚合起来。本人还处在学习期，并且百度的搜索一次只能搜一家网盘的结果，所以暂且还无法解决。如果以后有能力解决的话一定尽责更新。

轻松给你的WordPress加速——七牛云存储

发表于 2014-05-27   |   分类于 tech   |  

七牛云存储是目前国内功能和性能都非常不错的一家云存储提供商。其云存储产品采用网宿的CDN进行加速，加速节点遍布全国各地；支持图片裁剪，功能强大；价格也很不错，比阿里云OSS和又拍云要便宜些，每月提供10GB流量+10GB存储的免费套餐。

七牛云存储的镜像存储（源站管理）很有特色，其原理就是当云存储中不存在请求的文件时，云存储会到你指定的域名下寻找，并存储到云存储中。这样可以显著降低网站迁移成本，也免去了开发各程序相应插件的繁琐。只要你的资源文件使用独立域名，就可以非常简单地把资源文件迁移到七牛，帮助我们节省带宽和服务器的成本。

如何使用

登陆七牛，创建一个空间，然后点击 空间设置 > 一键加速，输入自己的域名。
