Struts2 上传示例

发表于 2016-02-20   |   分类于 coding   |  

使用Struts2上传组件非常方便

页面jsp：

<form action="fileUploadAction" enctype="multipart/form-data"
    method="POST">
    //enctype必须为"multipart/form-data；method必须为"POST"
    <input name="upfile" type="file"><br /> 
    //name很重要，定义后直接在java中调用
<input type="submit" value="上传">
    </button>
</form>

Action开发:

private String upfileFileName;//页面中定义的name：upfile，后加FileName
private File upfile;//页面中定义的name：upfile

public void setUpfileFileName(String upfileFileName) {
    this.upfileFileName = upfileFileName;
}

public String getUpfileFileName() {
    return upfileFileName;
}

public File getUpfile() {
    return upfile;
}
public void setUpfile(File upfile) {
    this.upfile = upfile;
}

@Override
public String execute(){
    String filePath = ServletActionContext.getServletContext().getRealPath("/upload");
    File fileSever = new File(filePath, upfileFileName);
    try {
        FileUtils.copyFile(upfile, fileSever);
    } catch (IOException e) {
        //异常操作
    }
    return SUCCESS;
}

这样上传功能就已经写好了，当然你会发现文件大小不能超过2M。这时需要在struts.xml中配置最大大小:

<struts>
    ...
    <constant name="struts.multipart.maxSize" value="10485760" />
    //value值:10M为 10*1024*1024
    ...
</struts>
